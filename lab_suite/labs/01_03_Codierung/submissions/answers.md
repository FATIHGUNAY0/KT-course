- Was zeigen die ausgegebenen Huffman-Codes?  
  *[kurz beschreiben]*
  Die Ausgabe zeigt, dass jedem Zeichen ein binärer Code zugeordnet wird. Häufig vorkommende Zeichen bekommen kürzere Codes, während seltene Zeichen längere Codes erhalten. Dadurch wird der Text effizienter codiert.

- Warum haben häufigere Zeichen kürzere Codewörter?  
  *[kurz begründen]*
  Häufige Zeichen bekommen kürzere Codes, damit die durchschnittliche Codelänge minimiert wird. So kann die gesamte Datenmenge reduziert werden, weil oft vorkommende Zeichen weniger Bits benötigen.
